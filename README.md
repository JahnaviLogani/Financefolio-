# FinanceFolio

## Overview

FinanceFolio is a personal finance management application built using Python and Streamlit. It helps users record daily expenses, categorize transactions, and analyze spending patterns through reports and visualizations.

This project is designed to provide a simple and user-friendly way to manage personal expenses and make better financial decisions.

## Features

* Add daily expenses
* View all recorded expenses
* Dashboard with expense summary
* Monthly and category-wise reports
* Interactive charts
* Download expense data as CSV

## Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

## Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/FinanceFolio.git
```

2. Navigate to the project directory.

```bash
cd FinanceFolio
```

3. Install the required dependencies.

```bash
pip install -r requirements.txt
```

4. Run the application.

```bash
streamlit run app.py
```

## Usage

* Add your daily expenses.
* View and search expense records.
* Analyze spending using the dashboard and reports.
* Export your expense data as a CSV file.
